# Box

The foundational layout primitive — a polymorphic `div` with every style token as a typed prop.

<ComponentPreview name="box-demo" />

## Import

```tsx
import { Box } from "astralis-ui";
```

## Usage

Box turns the design system's tokens into props: spacing, sizing, color,
borders, radius, shadows and positioning, all typed — your editor
autocompletes the valid values, and anything off-scale is a type error.
Every other layout component ([Flex](/docs/components/flex),
[Stack](/docs/components/stack), [Grid](/docs/components/grid), …) extends
Box, so everything on this page applies to all of them.

<ComponentPreview name="box-style-props" />

## Responsive props

Every style prop also accepts a breakpoint map with `base`, `sm`, `md`, `lg`
and `xl` keys — resolved to precompiled classes, no runtime style computation.

<ComponentPreview name="box-responsive" />

```tsx
<Box p={{ base: "4", md: "8", lg: "12" }} rounded={{ base: "md", md: "2xl" }} />
```

## Polymorphism

`as` swaps the rendered element — reach for it whenever the semantic element
matters (`section`, `aside`, `nav`, `figure` …).
