## Props

Code extends [Box](/docs/components/box).

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `variant` | `"subtle" \| "solid" \| "outline"` | `"subtle"` | Tinted, strongly tinted, or bordered — all track the active theme. |
| `size` | `"sm" \| "md" \| "lg"` | `"sm"` | Font size step. |
