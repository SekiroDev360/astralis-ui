| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `status` | `"info" \| "success" \| "warning" \| "error"` | `"info"` | Picks the icon, hue, and ARIA role. |
| `variant` | `"subtle" \| "solid" \| "outline" \| "left-accent"` | `"subtle"` | Fill treatment. |

## Accessibility

- `error` and `warning` render `role="alert"` (announced assertively);
  `info` and `success` render `role="status"` (announced politely).
- The dismiss button ships with an `aria-label`; the status icon is
  decorative (`aria-hidden`) because the role already conveys the meaning.
