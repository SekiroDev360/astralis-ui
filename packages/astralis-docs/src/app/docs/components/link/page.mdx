# Link

An inline text link with accent-channel coloring and external-link handling.

<ComponentPreview name="link-demo" />

## Import

```tsx
import { Link } from "astralis-ui";
```

For client-side routing, pass your router's component:
`<Link as={NextLink} href="/docs">`.

## Props
